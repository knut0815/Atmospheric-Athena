Atmospheric Athena Documentation
This work is a modified version of Athena hydrodynamics code (v4.1), extended to included ionizing radiative transfer from a planar source. This code is compatible with static mesh refinement and MPI parallelization. The physical system that the code models, and the equations that it solves, are described in Tripathi, Kratter, Murray-Clay, & Krumholz, ApJ, 2015.

These pages document the functions in our code. Of note, the src/ionradiation directory contains the radiative transfer specific functions and files. This page, under the Files tab, may be the most helpful because it lists all functions and variables, with links to the files they belong to.
