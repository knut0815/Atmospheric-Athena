Detailed Description

Integrate MHD equations using 3D version of the directionally unsplit MUSCL-Hancock (VL) integrator.

PURPOSE: Integrate MHD equations using 3D version of the directionally unsplit MUSCL-Hancock (VL) integrator. The variables updated are:

  • U.[d,M1,M2,M3,E,B1c,B2c,B3c,s] – where U is of type ConsS
  • B1i, B2i, B3i – interface magnetic field Also adds gravitational source terms, self-gravity, and the H-correction of Sanders et al.

For adb hydro, requires (9*Cons1DS + 3*Real + 1*ConsS) = 53 3D arrays

  • For adb mhd, requires (9*Cons1DS + 9*Real + 1*ConsS) = 80 3D arrays

REFERENCE:

  • J.M Stone & T.A. Gardiner, "A simple, unsplit Godunov method for multidimensional MHD", NewA 14, 139 (2009)
  • R. Sanders, E. Morano, & M.-C. Druguet, "Multidimensinal dissipation for upwind schemes: stability and applications to gas dynamics", JCP, 145, 511 (1998)
